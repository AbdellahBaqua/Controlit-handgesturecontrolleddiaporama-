Installation
============

Clone the repo:

.. code-block:: bash

    git clone https://github.com/yourusername/Controlit-handgesturecontrolleddiaporama-.git
    cd Controlit-handgesturecontrolleddiaporama-
    pip install -r requirements.txt

Usage
=====

Run the system:


    python app.py

Controls:

- `q` to quit
- `d` for debug
- `m` to toggle model (HOG/CNN)

Place a `professor.jpg` file with a clear face in the root folder.
